## Preparations required before running the code
Open RStudio and run the following command in the console to install renv:

```r
install.packages("renv")
```
Among renv is the officially recommended virtual environment tool by RStudio. 

And you can start with a project with Study on Youth Suicide.Rproj and renv.lock, restore the virtual environment as follows:
1. Open the MMM.Rproj project in RStudio (via .Rproj).

2. Run in the console:
```r
renv::restore()
```
This command will automatically install all dependencies and their specific versions as recorded in renv.lock, ensuring your environment matches the original’s.
